Utaka – A Cloud Storage Service

Utaka Core Functionality

After the request has been authorized by the ACL, the request will be given to the appropriate method inside one of the modules in the core. The core functions handle all additions, changes, and deletions to or from the database and filesystem. The core is divided into three modules: Service, Bucket, and Object.

  • Service
    • getService – Returns a list of buckets the user owns
  • Bucket
    • getBucket – Returns a list of objects the bucket contains
    • setBucket – Creates a new bucket
    • destroyBucket – Deletes a bucket
  • Object
    • getObject – Returns an object or just its database entries
    • setObject – Creates or overwrites an object
    • cloneObject – Creates a copy of an object
    • destroyObject – Deletes an object
