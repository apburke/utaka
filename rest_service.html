Utaka – A Cloud Storage Service

Utaka REST API – Service

GET Service

The GET method on the service will return a list of buckets owner by the request sender. It is currently the only REST method directed on the service.

Request

GET / HTTP/1.1
Host: file.hisg.org
Date: date
Authorization: signature

Response

HTTP/1.1 200 OK
Date: date
Content-Type: type
Content-Length: length
Connection: close
Server: HISGUtaka

<?xml version="1.0" encoding="UTF-8"?>
<ListAllMyBucketsResult xmlns="http://mattmillr.github.com/utaka">
  <Owner>
    <ID>user_id</ID>
    <DisplayName>display_name</DisplayName>
  </Owner>
  <Buckets>
    <Bucket>
      <Name>bucket_name</Name>
      <CreationDate>date</CreationDate>
    </Bucket>
    ...
  </Buckets>
</ListAllMyBucketsResult>

Headers/Elements

NameDescription
ListAllMyBucketsResult Root element.
Children: Owner and Buckets
Parent: None
Owner Contains information about the bucket's owner.
Children: ID and DisplayName
Parent: ListAllMyBucketsResult
ID The owner's unique ID.
Data: String (restricted to integers)
Parent: Owner
DisplayName The owner's unique name.
Data: String
Parent: Owner
Buckets Contains a list of buckets.
Children: Bucket
Parent: ListAllMyBucketsResult
Name The bucket's unique name.
Data: String
Parent: Bucket
CreationDate The date the bucket was created.
Data: String (Date formatted)
Parent: Bucket
